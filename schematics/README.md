# Схема адаптера токовой петли

![схема](schematic.png)

![плата](board.png)

## Отступления схемы от реального железа

1. Q2 установлен BD139 (может быть установлен MJE340)
2. Вместо D2 установлена пара из встречно включенных стабилитронов на 24 вольта (при установке 1n4004, не хватает быстродействия и телетайп не отпускает селекторный магнит достаточно быстро).

## Настройка

1. настроить повышающий преобразователь для питания токовой петли до ~24вольт
2. подключить преобразователь к разъёму SV1
3. замкнуть разъём токовой петли на TX (XOUT) через амперметр.
4. резистором R8 настроить рабочий ток в 40мА.
5. разомкнуть TX, замнуть RX (XIN) через амперметр
6. резистором R1 настроить рабочий ток в 40мА.
7. подключить к XOUT резистор соответствующий примерно сопротивлению катушки телетайпа.
8. контролируя ток через резистор - убавить напряжение на повышающем преобразователе токовой петли до начала снижения тока
9. слегка поднять напряжение на преобразователе.

Шаги 7-9 нужны для снижения рассеиваемой мощности на транзисторах регулятора тока.
